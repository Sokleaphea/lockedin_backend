# Tasks API Documentation

Base URL (local): `http://localhost:3000`

All endpoints are prefixed with `/api`.

## Data Model (Task)

```json
{
  "_id": "mongodb-objectid",
  "userId": "mongodb-objectid",
  "title": "string",
  "description": "string",
  "status": "pending" | "completed",
  "dueDate": "2026-02-03T00:00:00.000Z",
  "createdAt": "2026-02-03T12:00:00.000Z",
  "updatedAt": "2026-02-03T12:00:00.000Z"
}
```

> `_id` is auto-generated by MongoDB on creation.

---

## 1) Create Task

**POST** `/api/tasks`

### Request Body

```json
{
  "userId": "65c0f1234567890abcdef999",
  "title": "Finish report",
  "description": "Write the final report",
  "status": "pending",
  "dueDate": "2026-02-10"
}
```

### cURL

```bash
curl -X POST http://localhost:3000/api/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "65c0f1234567890abcdef999",
    "title": "Finish report",
    "description": "Write the final report",
    "status": "pending",
    "dueDate": "2026-02-10"
  }'
```

### Success Response (201)

```json
{
  "_id": "65c0f1234567890abcdef123",
  "userId": "65c0f1234567890abcdef999",
  "title": "Finish report",
  "description": "Write the final report",
  "status": "pending",
  "dueDate": "2026-02-10T00:00:00.000Z",
  "createdAt": "2026-02-03T12:00:00.000Z",
  "updatedAt": "2026-02-03T12:00:00.000Z",
  "__v": 0
}
```

## 4) Update Task (by \_id)

**PUT** `/api/tasks/<ID>`

### Request Body

```json
{
  "title": "Finish report (updated)",
  "status": "completed"
}
```

### cURL

```bash
curl -X PUT http://localhost:3000/api/tasks/65c0f1234567890abcdef123 \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Finish report (updated)",
    "status": "completed"
  }'
```

### Success Response (200)

```json
{
  "_id": "65c0f1234567890abcdef123",
  "userId": "65c0f1234567890abcdef999",
  "title": "Finish report (updated)",
  "description": "Write the final report",
  "status": "completed",
  "dueDate": "2026-02-10T00:00:00.000Z",
  "createdAt": "2026-02-03T12:00:00.000Z",
  "updatedAt": "2026-02-03T12:30:00.000Z",
  "__v": 0
}
```

---

## 5) Delete Task (by \_id)

**DELETE** `/api/tasks/<ID>`

### cURL

```bash
curl -X DELETE http://localhost:3000/api/tasks/65c0f1234567890abcdef123 \
  -H "Content-Type: application/json"
```

### Success Response (200)

```json
{
  "message": "Todo deleted"
}
```

---

## Error Responses

- 400: Validation errors (empty `title`, invalid `status`, invalid `dueDate`)
- 404: Task not found
- 500: Server error
